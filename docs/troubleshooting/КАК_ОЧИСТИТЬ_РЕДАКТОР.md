# Как очистить редактор Supabase SQL Editor

## Способ 1: Через клавиатуру (быстро)

1. Откройте Supabase SQL Editor
2. Нажмите **Ctrl+A** (или **Cmd+A** на Mac) - выделится весь текст
3. Нажмите **Delete** или **Backspace** - весь текст удалится
4. Редактор пустой ✅

## Способ 2: Через мышь

1. Откройте Supabase SQL Editor
2. Трижды кликните мышью в любом месте текста (тройной клик)
3. Или зажмите левую кнопку мыши и выделите весь текст
4. Нажмите **Delete** или **Backspace**

## Способ 3: Через меню (если есть)

1. Откройте Supabase SQL Editor
2. Найдите кнопку **"Clear"** или **"Очистить"** (обычно справа вверху)
3. Нажмите на неё

## Проверка что редактор пустой

После очистки:
- ✅ Редактор должен быть полностью пустым
- ✅ Не должно быть никакого текста
- ✅ Курсор должен быть в начале пустого редактора

## После очистки

1. Откройте файл **`ТОЛЬКО_ТАБЛИЦЫ_БЕЗ_ПОЛИТИК.sql`**
2. Скопируйте весь код (Ctrl+A, Ctrl+C)
3. Вставьте в **пустой** редактор (Ctrl+V)
4. Нажмите **Run**

## ⚠️ Важно

Если после очистки вы все еще видите ошибку про политики, значит:
- Политики уже существуют в базе данных (это нормально!)
- Просто **игнорируйте** эту ошибку
- Таблицы все равно создадутся
- Проверьте результат тестом: `bun test-registration.ts`
