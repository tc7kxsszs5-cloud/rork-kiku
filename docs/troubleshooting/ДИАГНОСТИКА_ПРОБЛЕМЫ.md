# Диагностика проблемы "только текущая страница"

## Что проверить в браузере:

### 1. Откройте консоль разработчика
- Нажмите `F12` или `Cmd+Option+I` (Mac)
- Перейдите на вкладку **Console**

### 2. Проверьте ошибки
- Есть ли красные ошибки в консоли?
- Какие именно ошибки?

### 3. Проверьте Network (Сеть)
- Перейдите на вкладку **Network**
- Обновите страницу (`F5` или `Cmd+R`)
- Есть ли запросы, которые не загружаются (красные)?

### 4. Проверьте React DevTools
- Установите расширение React DevTools для браузера
- Проверьте, какие компоненты рендерятся

## Возможные проблемы:

### Проблема 1: Застрял на экране выбора роли
**Решение:** 
- Нажмите на карточку "Родитель" или "Ребенок"
- Нажмите кнопку "Продолжить"
- Должен произойти переход на экран регистрации

### Проблема 2: Ошибка загрузки данных
**Решение:**
- Проверьте консоль на ошибки
- Убедитесь, что backend доступен: `https://backend-three-mauve-67.vercel.app`

### Проблема 3: Проблема с роутингом
**Решение:**
- Попробуйте открыть напрямую: `http://localhost:8081/role-selection`
- Или: `http://localhost:8081/register-parent`

## Быстрая проверка:

1. Откройте консоль браузера (`F12`)
2. Скопируйте все ошибки (если есть)
3. Проверьте, что видно на экране
4. Попробуйте кликнуть на элементы интерфейса

## Если видите только белый экран:

1. Обновите страницу (`F5`)
2. Очистите кэш браузера (`Cmd+Shift+R` на Mac)
3. Попробуйте в режиме инкогнито
4. Проверьте консоль на ошибки
