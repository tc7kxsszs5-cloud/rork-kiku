# Homebrew install прервался (SIGTERM / Tier 2)

Сборка LLVM/Ruby через Homebrew была завершена (SIGTERM). Ruby не установился.

---

## Вариант 1: Запустить установку снова и не прерывать

Установка может занять **до 1–2 часов**. Терминал не закрывать и не нажимать Ctrl+C.

```bash
rm -rf "$(brew --prefix)/var/homebrew/locks" 2>/dev/null
brew install ruby
```

Если есть возможность, запустите в **screen** или **tmux**, чтобы закрытие вкладки не убивало процесс:

```bash
brew install screen   # если нет
screen -S ruby
brew install ruby
# Отключиться: Ctrl+A, затем D. Подключиться снова: screen -r ruby
```

---

## Вариант 2: Обновить Homebrew — возможно, будет готовая сборка (bottle)

Иногда после обновления для вашей системы есть бинарный пакет Ruby без долгой сборки:

```bash
brew update
brew install ruby
```

Если в выводе будет «Pouring» (а не «Building») — ставится из бутылки, быстро.

---

## Вариант 3: Разработка без CocoaPods (пока)

Можно работать с приложением **без** `pod install` и без Xcode-сборки:

1. **На телефоне:** установите **Expo Go** из App Store. В проекте: `bun run start` → отсканируйте QR-код. Приложение откроется в Expo Go.
2. **В браузере:** `bun run start` → в меню нажмите **w** (web).

Так можно разрабатывать и тестировать. Сборка под iOS симулятор и `bun run ios` понадобятся позже, когда получится поставить Ruby и CocoaPods.

| Цель | Действие |
