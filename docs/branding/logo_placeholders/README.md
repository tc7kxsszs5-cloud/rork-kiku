## Instructions

### To create PNG files from SVG:

**Option 1: Online tool**
1. Visit https://svgtopng.com/
2. Upload `logo.svg`
3. Set size to 1024x1024
4. Download as `logo_color.png`

**Option 2: Inkscape (desktop)**
```bash
inkscape logo.svg --export-filename=logo_color.png --export-width=1024 --export-height=1024
```

**Option 3: ImageMagick**
```bash
convert -background none -size 1024x1024 logo.svg logo_color.png
```

### For white version:

Edit `logo.svg` - change all `fill="white"` to your desired color and `fill="#3B82F6"` to white, then export.
