## Do's and Don'ts

### Do's ‚úÖ
- Use rounded corners
- Use brand colors consistently
- Use child-friendly imagery
- Keep interface clean and uncluttered
- Test with children (usability)
- Prioritize safety messaging

### Don'ts ‚ùå
- Don't use sharp, aggressive shapes
- Don't use dark, gloomy colors
- Don't use complex language for children
- Don't clutter interface
- Don't use scary or anxiety-inducing imagery
- Don't forget accessibility (WCAG AA minimum)
