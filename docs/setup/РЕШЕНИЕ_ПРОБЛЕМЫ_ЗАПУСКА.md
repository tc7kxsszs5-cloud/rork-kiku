# Решение проблемы запуска приложения

## Проблема
```
CommandError: ngrok tunnel took too long to connect.
```

## Решения

### Вариант 1: Запуск без туннеля (рекомендуется)

```bash
bun run start:local
```

Или:

```bash
EXPO_USE_DEV_CLIENT=false bunx expo start --lan
```

### Вариант 2: Запуск для веб-браузера

```bash
bun run start:web
```

Или:

```bash
bunx expo start --web
```

### Вариант 3: Запуск с очисткой кэша

```bash
bun run start:clear
```

Или:

```bash
bunx expo start --lan --clear
```

### Вариант 4: Простой запуск без переменных окружения

```bash
bun run start:simple
```

## После запуска

1. Отсканируйте QR-код в Expo Go (для мобильного устройства)
2. Или откройте в браузере (если запустили `start:web`)

## Если проблема сохраняется

1. Проверьте интернет-соединение
2. Попробуйте перезапустить терминал
3. Очистите кэш: `bun run start:clear`
4. Проверьте, что порты не заняты

## Для тестирования регистрации

После запуска приложения:
1. При первом запуске должен появиться экран выбора роли
2. Выберите "Я родитель" → зарегистрируйтесь
3. Получите код и используйте его для регистрации ребенка
