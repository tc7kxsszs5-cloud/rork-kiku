# Как пользоваться Nanobot

Nanobot — это **веб-приложение**: ты запускаешь его в терминале, а общаешься с агентом в **браузере** по адресу http://localhost:8080. Отдельного десктопного приложения нет.

## Как это работает

1. **Терминал** — команда `nanobot run ...` поднимает локальный сервер на порту 8080.
2. **Браузер** — открываешь http://localhost:8080 и видишь чат с агентом (как в веб-мессенджере).
3. **Конфиг** — в `nanobot.yaml` указываешь, какой моделью пользоваться (OpenAI/Anthropic) и какие MCP-серверы подключить.

## Быстрый старт

### 1. Создай конфиг

В любой папке (например в домашней или в проекте) создай файл `nanobot.yaml`:

```yaml
agents:
  assistant:
    name: Помощник
    model: gpt-4o   # или claude-3-5-sonnet-20241022
    # mcpServers: []  # опционально: список MCP-серверов

# mcpServers: {}   # опционально: описание серверов
```

### 2. Задай API-ключ

В терминале (или в `~/.zshrc`):

```bash
# Для OpenAI (gpt-4o и т.п.)
export OPENAI_API_KEY="sk-..."

# Или для Anthropic (claude-*)
export ANTHROPIC_API_KEY="sk-ant-..."
```

### 3. Запусти Nanobot

```bash
cd ~   # или папка, где лежит nanobot.yaml
nanobot run ./nanobot.yaml
```

В терминале появится что-то вроде: `Listening on http://localhost:8080` или подобное.

### 4. Открой в браузере

Открой в Chrome/Safari/Firefox:

**http://localhost:8080**

Там будет интерфейс чата — пиши сообщения агенту, он отвечает через выбранную модель (и через MCP-серверы, если ты их указал в конфиге).

### 5. Остановка

В терминале, где запущен `nanobot run`, нажми **Ctrl+C** — сервер остановится, страница в браузере перестанет работать.

## Итого

| Вопрос | Ответ |
|--------|--------|
| Есть ли отдельное приложение? | Нет, только веб-интерфейс на localhost:8080. |
| Где запускать? | В терминале: `nanobot run ./nanobot.yaml`. |
| Где общаться с агентом? | В браузере: http://localhost:8080. |
| Нужен ли конфиг? | Да, минимум `nanobot.yaml` с агентом и моделью. |
| Нужен ли API-ключ? | Да, OpenAI или Anthropic, в переменной окружения. |

Подробнее: [документация Nanobot](https://docs.nanobot.ai), [примеры конфигов](https://github.com/nanobot-ai/nanobot/tree/main/examples).
