# Альтернативные способы запуска

## Если `bun run start:web` не работает:

### Способ 1: Прямой запуск Expo

```bash
cd /Users/mac/Desktop/rork-kiku
bunx expo start --web
```

### Способ 2: С указанием порта

```bash
bunx expo start --web --port 8081
```

### Способ 3: Без переменных окружения

```bash
EXPO_NO_DOTENV=1 bunx expo start --web
```

### Способ 4: С очисткой кэша

```bash
bunx expo start --web --clear
```

## После запуска:

1. Дождитесь сообщения: "Metro waiting on..."
2. В консоли появится ссылка типа: `http://localhost:8081`
3. Откройте эту ссылку в браузере вручную

## Если не открывается автоматически:

1. Скопируйте URL из консоли (например: `http://localhost:8081`)
2. Вставьте в адресную строку браузера
3. Нажмите Enter

## Проверка что работает:

После открытия в браузере вы должны увидеть:
- Экран выбора роли (если первый запуск)
- Или главный экран приложения (если уже зарегистрированы)

## Если все еще не работает:

1. Проверьте, что порт свободен: `lsof -i:8081`
2. Попробуйте другой порт: `bunx expo start --web --port 8082`
3. Откройте вручную: `http://localhost:8082`
