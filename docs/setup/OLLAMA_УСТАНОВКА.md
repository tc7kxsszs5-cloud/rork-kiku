# Установка Ollama — AI без API и без VPN

**Ollama** — программа для Mac, которая запускает нейросети **у тебя на компьютере**. Никакой регистрации, API-ключей и VPN не нужно. Всё работает локально.

## Установка

### Вариант 1: Скачать с сайта (проще всего)

1. Открой в Safari: **https://ollama.com/download/mac**
2. Скачай установщик (файл `.dmg`).
3. Открой скачанный файл и перетащи **Ollama** в папку **Programs** (Приложения).
4. Запусти Ollama из папки Programs — в строке меню появится иконка Ollama (лампа/кит). Сервер работает в фоне.

### Вариант 2: Через терминал (Homebrew)

Если установлен Homebrew:

```bash
brew install ollama
```

Потом запусти Ollama (оставь терминал открытым или запусти из Programs, если ставил через .dmg):

```bash
ollama serve
```

## Как пользоваться

1. **Терминал** — открой Терминал и введи, например:
   ```bash
   ollama run llama3.2
   ```
   Первый раз Ollama скачает модель (несколько гигабайт), потом откроется чат в терминале. Пиши сообщения и нажимай Enter.

2. **Веб-интерфейс** — если хочешь чат в браузере как у Nanobot:
   - Убедись, что Ollama запущена (иконка в меню или `ollama serve` в терминале).
   - Открой в Safari: **http://localhost:11434**
   - Или установи отдельный UI, например **Open WebUI** (опционально).

## Популярные модели

| Команда | Модель | Размер |
|--------|--------|--------|
| `ollama run llama3.2` | Llama 3.2 | ~2 GB |
| `ollama run mistral` | Mistral | ~4 GB |
| `ollama run phi3` | Phi-3 (легкая) | ~2 GB |
| `ollama run gemma2` | Gemma 2 | ~5 GB |

Полный список: **https://ollama.com/library**

## Требования

- macOS 14 (Sonoma) или новее (для установки с сайта).
- Свободное место на диске: от 2–3 GB для лёгких моделей до 10+ GB для больших.

## Итого

| | Nanobot | Ollama |
|---|--------|--------|
| Нужен API-ключ? | Да (OpenAI/Anthropic) | Нет |
| Нужен VPN? | Часто да (если блокируют) | Нет |
| Где считает? | В облаке | На твоём Mac |
| Регистрация? | Да (на сайте провайдера) | Нет |

Если блокируют и OpenAI, и Anthropic — **Ollama** даёт возможность работать с AI без VPN и без ключей.
