## 1. Leadership

### Founder / CEO
**Ответственность**:
- Product vision и strategy
- Fundraising и investor relations
- Partnerships (schools, NGOs)
- Legal и compliance oversight
- Team building

**Skills**: Product management, business development, fundraising

### CTO (Chief Technology Officer)
**Ответственность**:
- Technical architecture decisions
- Engineering team leadership
- Infrastructure и DevOps strategy
- Security oversight
- ML/AI strategy

**Skills**: Full-stack development, cloud architecture, ML/AI

### CPO (Chief Product Officer)
**Ответственность**:
- Product roadmap
- User research и UX design
- Feature prioritization
- Analytics и metrics
- User feedback integration

**Skills**: Product design, UX/UI, user research

## 2. Engineering

### Mobile Engineer (iOS)
**Ответственность**:
- iOS app development (React Native + Expo)
- UI/UX implementation
- Performance optimization
- App Store submission
- Bug fixes и testing

**Skills**: React Native, TypeScript, Expo, iOS development

### Mobile Engineer (Android)
**Ответственность** (Phase 2):
- Android app development
- Cross-platform coordination с iOS
- Google Play submission

**Skills**: React Native, Android development

### Backend Engineer
**Ответственность**:
- API development (tRPC/REST)
- Database design и optimization
- Authentication и authorization
- Integration с third-party services
- Performance и scalability

**Skills**: Node.js, TypeScript, PostgreSQL, Redis, AWS

### ML Engineer
**Ответственность**:
- ML model training и deployment
- Image moderation pipeline
- Model accuracy monitoring
- Retraining и optimization
- Research new ML approaches

**Skills**: Python, PyTorch/TensorFlow, Computer Vision, MLOps

### SRE / DevOps Engineer
**Ответственность**:
- Infrastructure as Code (Terraform)
- CI/CD pipelines (GitHub Actions)
- Kubernetes management
- Monitoring и alerting
- Incident response

**Skills**: Kubernetes, Docker, Terraform, AWS, monitoring tools
