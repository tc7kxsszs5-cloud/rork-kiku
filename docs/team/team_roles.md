## Core Team Roles

- React Native/native development
- Mobile app architecture
- Performance optimization
- App Store submissions

- **Team:** 2-3 mobile developers

**Backend Lead:**
- API design и implementation
- Microservices architecture
- Database schema design

**DevOps/SRE:**
- CI/CD pipelines
- Kubernetes administration
- Monitoring и alerting
- Incident response
- Cost optimization
- **Team:** 1 SRE (Year 1), scale to 2-3

**QA Engineer:**
- Test automation
- Manual testing
- Bug tracking
- Release verification
- Performance testing
- **Team:** 1-2 QA engineers
