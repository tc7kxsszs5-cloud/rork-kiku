# Data Room Checklist для Rork-Kiku

## Обзор

Data room — это secure repository документов, который предоставляется потенциальным инвесторам, acquirers или partners для due diligence. Этот чек-лист охватывает все документы, которые должны быть подготовлены и организованы.

### Платформы для Data Room

**Опции:**
1. **DocSend** — популярный для startups, analytics on viewing
2. **Dropbox** — простой, знакомый, но меньше security
3. **Google Drive** — бесплатный, но меньше professional
4. **Carta** — если используете для cap table, интегрированный data room
5. **DealRoom, Intralinks** — enterprise-grade, дороже

**Рекомендация для Seed stage:** DocSend или Dropbox Business

### Access Control

- **Tiered access:** Не все investors получают доступ ко всему сразу
  - **Tier 1 (Initial interest):** Pitch deck, one-pager
  - **Tier 2 (Serious interest):** Financial model, product demo, key contracts
  - **Tier 3 (Term sheet stage):** Full data room access
- **NDAs:** Все investors подписывают NDA перед доступом к data room
- **Watermarks:** Документы с watermarks (имя investor, дата)
- **Tracking:** Мониторинг кто что просматривает (DocSend analytics)

### Maintenance

- **Update regularly:** Как минимум quarterly или перед fundraising
- **Version control:** Dates на всех документах, clear versioning
- **Q&A log:** Track вопросы от investors и ответы
- **Redactions:** Sensitive info (SSNs, account numbers) redacted где appropriate

## Timeline для подготовки Data Room

**8 weeks до начала fundraising:**

- Week 1-2: Инвентаризация существующих документов
- Week 3-4: Создание missing documents (policies, compliance docs)
- Week 5-6: Legal review всех документов
- Week 7: Организация в data room structure
- Week 8: Final review, access testing, NDA preparation

**Ongoing:** Update после major events (new hire, contract signed, etc.)

## Checklist использования

- [ ] Пройтись по всем категориям
- [ ] Mark что есть, что нужно создать
- [ ] Prioritize: Must-have vs Nice-to-have для вашего stage
- [ ] Assign owners для создания missing docs
- [ ] Set deadlines
- [ ] Review и finalize
- [ ] Organize в data room platform
- [ ] Test access
- [ ] Brief team на что investors будут смотреть

**Note:** Не все документы applicable на pre-seed/seed stage. Фокус на:
1. Corporate basics (registration, cap table)
2. IP assignments
3. Employment agreements
4. Privacy/ToS
5. Financial model
6. Product/tech docs

Остальное добавляется по мере роста компании.
