## Введение

Добро пожаловать в Rork-Kiku! Мы — платформа для безопасного общения детей и родителей с проактивной модерацией контента.

**Конфиденциальность и безопасность ваших детей — наш приоритет #1.** Эта политика конфиденциальности объясняет:
- Какие данные мы собираем
- Как мы используем эти данные
- Как мы защищаем данные ваших детей
- Ваши права как родителя

**Важно**: Эта политика применяется к:
- **Родителям** (18+ лет)
- **Детям** (6-12 лет, с согласия родителя)

Мы соблюдаем:
- **COPPA** (Children's Online Privacy Protection Act) — США
- **GDPR** (General Data Protection Regulation) — ЕС
- **CCPA** (California Consumer Privacy Act) — Калифорния

## 1. Информация, которую мы собираем

### 1.1 Информация о родителях (18+ лет)

**При регистрации**:
- Имя
- Email address
- Apple ID (если регистрация через Apple Sign In)
- Номер телефона (опционально)

**При использовании приложения**:
- IP address
- Device information (модель, OS version)
- Usage data (время в приложении, features used)
- Payment information (для premium подписки)
  - Обрабатывается через Stripe/Apple Pay (мы НЕ храним полные данные карт)

### 1.2 Информация о детях (6-12 лет) — с согласия родителя

**С согласия родителя мы собираем**:
- Имя ребёнка (только имя, без фамилии)
- Дата рождения (для возрастной проверки)
- Аватар (опционально, загруженное фото)
- Контент, созданный ребёнком:
  - Текстовые сообщения
  - Фотографии
  - Видео (в будущем)

**Мы НЕ собираем от детей**:
- ❌ Фамилию
- ❌ Адрес проживания
- ❌ Номер телефона
- ❌ School name или location
- ❌ Geolocation (GPS координаты)
- ❌ Social Security Number
- ❌ Любые другие PII (Personally Identifiable Information)

**COPPA Compliance**:
- Мы собираем **минимум данных**, необходимых для работы сервиса
- Родитель даёт **явное согласие** перед созданием детского профиля
- Родитель может в любой момент **удалить данные ребёнка**

### 1.3 Автоматически собираемая информация

**Log Data**:
- IP address
- Browser type
- Device ID
- Time stamps (когда вы использовали приложение)
- Pages viewed (какие экраны открывали)

**Cookies**:
- Мы используем cookies для:
  - Сессии (чтобы вы оставались logged in)
  - Analytics (Google Analytics, Mixpanel)
  - Security (CSRF protection)
- Вы можете отключить cookies в настройках браузера (но функционал может быть ограничен)

## 2. Как мы используем собранную информацию

### 2.1 Для родителей

- **Предоставление сервиса**: регистрация, аутентификация, доступ к приложению
- **Коммуникация**: email уведомления (новости, updates, moderation alerts)
- **Billing**: обработка платежей за premium подписку
- **Analytics**: улучшение продукта, понимание usage patterns
- **Support**: ответы на ваши вопросы и проблемы
- **Security**: защита от fraud, abuse

### 2.2 Для детей

- **Предоставление сервиса**: отправка/получение сообщений, доступ к контенту
- **Content Moderation**: проверка контента на безопасность (AI + human moderation)
- **Parental Dashboard**: родители могут видеть активность ребёнка
- **Analytics**: агрегированная статистика (не individual tracking)

**Важно**: Мы **НЕ используем данные детей для**:
- ❌ Таргетированной рекламы
- ❌ Продажи данных третьим лицам
- ❌ Profiling для маркетинга
- ❌ Behavioral advertising

## 3. Как мы делимся информацией

### 3.1 Мы НЕ продаём ваши данные

**Ваши данные и данные ваших детей НЕ продаются, НЕ арендуются, НЕ передаются третьим лицам для маркетинга.**

### 3.2 Мы делимся информацией только в следующих случаях:

**Service Providers** (необходимые для работы сервиса):
- **Hosting**: AWS / GCP (хранение данных, серверы)
- **Authentication**: Apple Sign In (только для аутентификации)
- **Payment Processing**: Stripe / Apple Pay (обработка платежей)
- **ML Moderation**: Google Vision API (для проверки изображений)
- **Analytics**: Google Analytics, Mixpanel (агрегированные данные)
- **Email**: SendGrid / AWS SES (email уведомления)

**Все service providers подписывают Data Processing Agreements (DPA) и соблюдают GDPR/COPPA.**

**Legal Obligations**:
- Если требуется по закону (court order, subpoena)
- Для защиты прав и безопасности (например, отчёт о CSAM в NCMEC)
- В случае расследования преступлений

**Business Transfers**:
- В случае слияния, поглощения или продажи компании (данные могут быть переданы новому владельцу)
- Мы уведомим вас заранее

**С вашего согласия**:
- Если вы явно попросите нас поделиться данными (например, экспорт в другой сервис)

## 4. Как мы защищаем ваши данные

### 4.1 Технические меры

**Encryption**:
- **In Transit**: все данные передаются через HTTPS/TLS 1.3
- **At Rest**: все данные шифруются на серверах (AES-256)
- **Media Files**: фото и видео шифруются в S3 с server-side encryption

**Access Control**:
- Ограниченный доступ к данным (только authorized personnel)
- Multi-factor authentication (MFA) для админов
- Role-Based Access Control (RBAC)

**Monitoring**:
- 24/7 мониторинг безопасности (SIEM)
- Intrusion detection
- Automated alerts для suspicious activity

**Backups**:
- Регулярные backups (daily)
- Encrypted backups
- Disaster recovery plan

### 4.2 Организационные меры

**Employee Training**:
- Все сотрудники проходят training по безопасности данных
- Background checks для сотрудников с доступом к данным
- NDA (Non-Disclosure Agreements)

**Security Audits**:
- Регулярные penetration tests
- Third-party security audits (ежегодно)
- Vulnerability scanning

**Incident Response**:
- Процедуры для data breaches
- Уведомление пользователей в течение 72 часов (GDPR requirement)
- Cooperation с правоохранительными органами

## 5. Права родителей (COPPA)

Как родитель, вы имеете следующие права в отношении данных вашего ребёнка:

### 5.1 Право на доступ

**Вы можете**:
- Просматривать все данные вашего ребёнка в Parent Dashboard
- Экспортировать данные (JSON или CSV format)
- Запросить копию всех данных (через support)

**Как**: Откройте Parent Dashboard → Settings → Export Data

### 5.2 Право на удаление

**Вы можете**:
- Удалить профиль ребёнка и все его данные
- Удалить отдельные сообщения или медиа
- Удалить всю учётную запись (включая свою)

**Как**: Parent Dashboard → Child Profile → Delete Profile

**Что происходит**:
- Все данные ребёнка удаляются с серверов (в течение 30 дней)
- Backups удаляются (в течение 90 дней)
- Данные нельзя восстановить после удаления

### 5.3 Право на контроль

**Вы можете**:
- Изменять настройки конфиденциальности ребёнка
- Менять уровень модерации (строгая/средняя/мягкая)
- Отключать функции (например, отправка видео)
- Блокировать контакты

**Как**: Parent Dashboard → Child Settings → Privacy & Moderation

### 5.4 Право на rectification (исправление)

**Вы можете**:
- Исправить информацию о ребёнке (имя, дата рождения)
- Обновить настройки

**Как**: Parent Dashboard → Edit Child Profile

### 5.5 Право отозвать согласие

**Вы можете**:
- В любой момент отозвать согласие на обработку данных ребёнка
- При отзыве согласия — профиль ребёнка будет удалён

**Как**: Parent Dashboard → Revoke Consent → Confirm

## 6. Права GDPR (для пользователей из ЕС)

Если вы находитесь в Европейском Союзе, у вас есть дополнительные права:

### 6.1 Right to Access

Вы можете запросить копию всех ваших данных (и данных ребёнка).

### 6.2 Right to Rectification

Вы можете исправить неточные данные.

### 6.3 Right to Erasure ("Right to be Forgotten")

Вы можете запросить удаление всех данных.

### 6.4 Right to Data Portability

Вы можете экспортировать данные в machine-readable format (JSON, CSV).

### 6.5 Right to Object

Вы можете возразить против обработки данных (например, для marketing).

### 6.6 Right to Restrict Processing

Вы можете попросить ограничить обработку данных.

**Как осуществить права**: Напишите на [PRIVACY_EMAIL] или через Parent Dashboard → Privacy Rights.

**Мы ответим в течение 30 дней** (GDPR requirement).

## 7. Хранение данных

### 7.1 Как долго мы храним данные

**Активные аккаунты**:
- Данные хранятся пока аккаунт активен
- Неактивные аккаунты (> 2 года без login) могут быть удалены (с уведомлением)

**Удалённые аккаунты**:
- Данные удаляются с серверов в течение 30 дней
- Backups удаляются в течение 90 дней

**Logs**:
- Security logs: 1 год
- Application logs: 30 дней
- Audit logs (COPPA): 2 года

**Legal Hold**:
- Если данные требуются для расследования или судебного процесса, мы можем сохранить их дольше
