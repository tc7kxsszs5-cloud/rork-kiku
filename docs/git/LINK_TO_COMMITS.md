# 🔗 Прямая ссылка на коммиты

## 📋 Ссылка на страницу коммитов

**Скопируйте и откройте в браузере:**

```
https://github.com/tc7kxsszs5-cloud/rork-kiku/commits/main
```

---

## 🎯 Что вы увидите

На этой странице будет:

- **Список всех коммитов** вашего проекта
- **Ваши последние коммиты:**
  - "Добавил инструкции и руководства"
  - "Описание ваших изменений"
- **Автор** каждого коммита
- **Время** создания коммита
- **Статусы проверок** (крестики ❌ или галочки ✅)
- **Изменения** в каждом коммите

---

## 💡 Как открыть

1. **Скопируйте** ссылку выше
2. **Вставьте** в адресную строку браузера
3. **Нажмите** Enter
4. **Увидите** все коммиты!

3. Нажмите на нее

## 🔍 Что искать на странице коммитов

### Структура страницы:

```
┌─────────────────────────────────────────┐
│  Коммиты (Commits)                     │
├─────────────────────────────────────────┤
│  🔄 "Добавил инструкции..."           │  ← Ваш коммит
│     автор • время • [статусы]          │
├─────────────────────────────────────────┤
│  🔄 "Описание ваших изменений"        │  ← Ваш коммит
│     автор • время • [статусы]          │
├─────────────────────────────────────────┤
│  ... другие коммиты ...                 │
└─────────────────────────────────────────┘
```

### Элементы коммита:

- **Сообщение коммита** - описание что было сделано
- **Автор** - кто создал коммит
- **Время** - когда был создан
- **Статусы** - результаты проверок (❌ или ✅)
- **ID коммита** - уникальный идентификатор
