# Privacy Policy - KIKU

**Last Updated:** January 2026

## Introduction

KIKU ("Kids Safety App", "we", "us", or "our") is committed to protecting the privacy of children and families. This Privacy Policy explains how we collect, use, disclose, and safeguard information when you use our mobile application.

## Compliance

This application is designed to comply with:
- **COPPA (Children's Online Privacy Protection Act)** - United States
- **GDPR (General Data Protection Regulation)** - European Union
- **GDPR-K (GDPR for Children)** - Enhanced protections for minors
- **Texas SCOPE Act** - Texas state regulations for child safety

## Information We Collect

### Information Collected Locally
All data is stored locally on your device using secure storage mechanisms:

1. **User Profile Data**
   - Name
   - Role (parent/child)
   - Email (optional)
   - Age (for child accounts)
   - Language preference
   - Device ID

2. **Monitoring Data**
   - Chat messages (for AI analysis)
   - Risk assessment results
   - Alert history
   - Contact lists
   - Time restriction settings

3. **Location Data**
   - GPS coordinates (only when SOS feature is activated)
   - Location data is not continuously tracked

4. **Authentication Data**
   - PIN (stored securely, hashed)
   - Biometric authentication settings

### Information Not Collected
- We do NOT collect personal data without verifiable parental consent
- We do NOT sell or rent children's personal information
- We do NOT share information with third parties without consent
- We do NOT use children's data for targeted advertising

## How We Use Information

### Primary Uses
1. **Safety Monitoring**: AI analysis of messages to detect threats, bullying, and inappropriate content
2. **Parental Controls**: Enable parents to set time limits, content filters, and contact restrictions
3. **Emergency Response**: SOS alerts with location sharing to guardians
4. **Educational Content**: Age-appropriate recommendations and safety tips

### Data Processing
- All AI analysis happens locally or through encrypted channels
- Messages are analyzed for safety patterns, not stored permanently
- Risk assessments are kept in local logs with automatic cleanup after 90 days

## Parental Consent

### COPPA Compliance
For children under 13 years old:
1. **Initial Consent**: Parents must create an account and explicitly consent to data collection
2. **Review Rights**: Parents can review all data collected about their child
3. **Deletion Rights**: Parents can request deletion of their child's data at any time
4. **Notification**: Parents are notified of any material changes to data practices

### Consent Management
- All settings changes require parent authentication
- Consent logs are maintained with timestamps
- Parents can withdraw consent and delete all data

## Data Security

### Security Measures
1. **Local Storage**: Data stored using AsyncStorage with encryption
2. **Secure Storage**: Sensitive data (PINs, credentials) use expo-secure-store
3. **Biometric Protection**: Face ID/Touch ID support for sensitive operations
4. **Session Management**: Automatic logout after inactivity
5. **End-to-End Encryption**: Messages analyzed through encrypted channels

### Data Retention
- User profiles: Until account deletion
- Chat analysis: 90 days automatic cleanup
- Compliance logs: 3 years (legal requirement)
- SOS alerts: 1 year for safety review

## User Rights

### Rights Under GDPR
1. **Right to Access**: Request a copy of all personal data
2. **Right to Rectification**: Correct inaccurate information
3. **Right to Erasure**: Delete all personal data ("right to be forgotten")
4. **Right to Restriction**: Limit processing of personal data
5. **Right to Portability**: Export data in a machine-readable format
6. **Right to Object**: Object to data processing

### Rights Under COPPA
1. **Review**: Review personal information collected from children
2. **Delete**: Request deletion of children's personal information
3. **Refuse Further Collection**: Stop further collection or use of children's data
4. **Notification**: Receive notice of data practices before collection

### Exercising Rights
To exercise any rights, contact us at: privacy@kiku-app.com
We will respond within 30 days.

## Age-Appropriate Content

### Age Groups
Content filtering is automatically adjusted based on child's age:

1. **Early Childhood (3-6 years)**
   - Strictest filtering
   - Educational content only
   - Maximum supervision

2. **Middle Childhood (7-9 years)**
   - High filtering
   - Age-appropriate learning
   - Parental oversight

3. **Pre-Teen (10-12 years)**
   - Moderate filtering
   - Increasing independence
   - Parent notifications

4. **Teen (13-17 years)**
   - Balanced filtering
   - Privacy-aware monitoring
   - Safety alerts only
