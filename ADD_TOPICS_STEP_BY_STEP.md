# 🏷️ Пошаговая инструкция: Как добавить Topics

## ✅ Способ через Settings (самый надежный)

Если не видите раздел About или Topics там нет, используйте Settings:

### Шаг 1: Откройте Settings напрямую

**Прямая ссылка:**
https://github.com/tc7kxsszs5-cloud/rork-kiku/settings

Или:
1. Откройте: https://github.com/tc7kxsszs5-cloud/rork-kiku
2. Нажмите на вкладку **"Settings"** (вверху страницы, рядом с Code, Issues, Pull Requests, Actions, Projects, Wiki, Security, Insights)
3. **Важно**: Вы должны быть владельцем репозитория или иметь права администратора!

### Шаг 2: Перейдите в раздел "General"

В левом меню Settings (боковая панель слева) нажмите на **"General"** - это первый пункт в списке.

Если не видите меню слева:
- Убедитесь, что вы на странице Settings
- Попробуйте увеличить окно браузера
- Попробуйте другой браузер

### Шаг 3: Прокрутите вниз до "Topics"

На странице General прокрутите вниз. Там будет несколько разделов:
- Repository name
- Description
- Website
- Topics ← **ВОТ ОНО!**
- ...

### Шаг 4: Добавьте теги

В поле "Topics" начните вводить теги. GitHub будет предлагать автодополнение.

**Список тегов для добавления:**
- react-native
- expo
- typescript
- child-safety
- ai
- machine-learning
- parental-control
- child-protection
- cyber-safety
- mobile-app
- safety
- security
- children
- parenting
- education
- gamification
- open-source

**Как добавлять:**
1. Начните вводить тег (например, "react-native")
2. GitHub предложит варианты - выберите нужный
3. Нажмите Enter или кликните на вариант
4. Тег добавится. Начните вводить следующий
5. Повторите для всех тегов

### Шаг 5: Сохраните

После добавления всех тегов:
1. Прокрутите страницу вниз
2. Нажмите кнопку **"Save changes"** (зеленая кнопка внизу страницы)
3. Или просто нажмите Enter в поле Topics - изменения сохранятся автоматически

---

## 🔍 Если все еще не видите Topics

### Возможные причины:

1. **Недостаточно прав** - вы не являетесь владельцем/администратором репозитория
   - Проверьте, что вы вошли в аккаунт `tc7kxsszs5-cloud`
   - Убедитесь, что у вас есть права администратора

2. **Другая версия интерфейса**
   - Попробуйте обновить страницу (Ctrl+R / Cmd+R)
   - Попробуйте другой браузер (Chrome, Firefox, Safari)

3. **Мобильная версия**
   - Переключитесь на desktop версию
   - Внизу страницы GitHub обычно есть ссылка "Desktop version"

4. **Поле скрыто**
   - Попробуйте использовать поиск на странице (Ctrl+F / Cmd+F)
   - Введите "Topics" в поиск

## 💡 Альтернативный способ (если через Settings не работает)

Если по какой-то причине Topics недоступен через Settings, можно попробовать через API, но это сложнее. Проще всего - связаться с поддержкой GitHub или убедиться, что вы имеете права администратора.

## ✅ Проверка после добавления

После добавления Topics:
1. Вернитесь на главную страницу репозитория
2. Topics должны отображаться в разделе About (справа)
3. Теперь репозиторий будет появляться в поиске по этим тегам!
